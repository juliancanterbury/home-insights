# Home Insights

Prototype GitHub Pages app for the Home Insights / house digital twin project.

## Deploy

1. Create a GitHub repository called `home-insights`.
2. Upload these files to the root of the repository.
3. Go to Settings → Pages.
4. Set Source to `Deploy from a branch`.
5. Select `main` and `/root`.
6. Open: `https://juliancanterbury.github.io/home-insights/`
